My coursework for MIT OCW 6.042J Mathematics For Computer Science Fall 2010 class. This covers the exact same coursework required of MIT students present at class for full marks.

All solutions are my own.

Taught by [Prof. Tom Leighton](https://ocw.mit.edu/search/?q=Prof.+Tom+Leighton) and [Dr. Marten van Dijk](https://ocw.mit.edu/search/?q=Dr.+Marten+van+Dijk).

Course Link: https://ocw.mit.edu/courses/6-042j-mathematics-for-computer-science-fall-2010/

## Topics Covered

| Topic | Subtopics |
|-------|-----------|
| Proofs | Introduction, induction, strong induction |
| Number Theory | GCD, modular arithmetic, Euler's theorem, RSA encryption |
| Graph Theory | Coloring, matching problems, minimum spanning trees, communication networks |
| Relations and Orders | Partial orders, scheduling |
| Counting and Asymptotics | Sums, asymptotics, divide and conquer recurrences, linear recurrences, counting rules |
| Probability | Introduction, conditional probability, independence, random variables, expectation, large deviations, random walks |
